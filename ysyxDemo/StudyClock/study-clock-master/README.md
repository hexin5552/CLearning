# Study Clock — 学习记录时钟

> "一生一芯"组队学习 Stage 1 实践项目

一个终端小工具，帮你养成学习记录的习惯：

- **回顾** — 每次启动时翻出你几天前写的学习笔记，提醒你回顾
- **记录** — 写下今天的学习感悟，自动保存到文件
- **计时** — 启动正计时器，记录你这次专注了多久

## 快速开始

```bash
make          # 编译
make run      # 编译并运行
make fancy    # 大字体模式（完成选做扩展后可用）
make clean    # 清理编译产物
```

> 需要 Linux / macOS 环境。Windows 用户请先配置 WSL，详见实现指南。

## 项目结构

```
study-clock/
├── csrc/

│   ├── record.c       ← 链表操作
│   ├── storage.c      ← 文件读写
│   ├── timer.c        ← 计时器
│   ├── display.c      ← 大字体显示（选做）
│   └── tools/term.c   ← 终端工具（框架）
├── include/           ← 头文件
├── data/

│   └── records.txt    ← 运行时生成

└── README.md
```

## 这个项目怎么做？

项目采用**骨架填空**的方式：框架代码（状态机主循环、终端控制、颜色宏等）已经写好，你需要实现标注了 `TODO` 的函数。

**推荐顺序**：`record.c` → `storage.c` → `timer.c` → `main.c`

详细的实现引导请阅读 **实现指南**（由助教单独发放）。

## 涉及的知识点

本项目覆盖 **Linux C 一站式** 第 1-9、11-16、21、23-25、26.1 章的核心内容：

| 知识点 | 在项目中的体现 |
|-------|--------------|
| 结构体（Ch7） | `Record` 链表节点 |
| 指针 / 二级指针（Ch23） | 链表头插、尾插 |
| 链表（Ch26.1） | 学习记录的内存组织 |
| 文件 I/O（Ch25.2） | 读写 `data/records.txt` |
| 字符串处理（Ch8, Ch25.1） | 解析文件格式、处理用户输入 |
| 预处理器（Ch21） | ANSI 颜色宏 |
| switch 分支（Ch4） | 状态机主循环 |
| 动态内存（Ch25.4） | `malloc` / `free` |
